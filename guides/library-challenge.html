Library Book Challenge

Supplies

  • Library cards if available
  • Reusable bag
  • This mission list

Instructions

  1. Each child finds: one funny book, one nonfiction book, one book with an animal, one book chosen only by its cover and one book to read together.
  2. The 3-year-old may choose picture books; older kids can help locate sections.
  3. Sit together for a ten-minute preview before checking out.
  4. At home, each child gives a one-sentence pitch for the first book to read.

Safety and setup notes

  • Follow current library checkout limits and return dates.
